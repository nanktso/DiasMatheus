Configurar Caminho Model Builder #PY

Código utilizado no Model Builder do ArcGIS para caso determinado output não for selecionado a ferramenta não realizar o processamento correspondente a esse output. 

    def pergunta(par1,par2):
        import arcpy
        if par1 == "false" and par2=="false":
            return "false"
        elif par1 == "true" and par2=="false":
            return "true"
        elif par1 == "false" and par2=="true":
            return "true"
        elif par1 == "true" and par2=="true":
            return "true"

Classificar Tamanho do Pixel #PY

Código utilizado no Model Builder do ArcGIS para classificar o tamanho dos pixels. As funções serão utilizadas a posteriori em outras ferramentas que irão manipular os pixels e necessitam dessa informação para rodarem. 

    def pixel_ate_20(par):
        import arcpy
        if par <= 20:
            return "true"
        else:
            return "false"

    def maior_que_20(par):
        import arcpy
        if par > 20:
            return "true"
        else:
            return "false"
